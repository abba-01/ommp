### **Updated Complete Classification Structure**

```
MMP Code = [Body][Observer Position & Altitude][Meta-domain][Relative Altitude][Kinematic State][Relative Speed][Spatial Morphology][Platform Modifier & Size]
```

### **Platform Modifier & Size Encoding**

The **Platform** segment explicitly encodes information about the conveyance or observer platform, specifying its relative size using modifiers:

* **Modifiers:**

  * `+`: Larger than observer’s own platform
  * `-`: Smaller than observer’s own platform
  * `=`: Matching the observer’s own platform size exactly (baseline)

* **Size Measurement (`x.x.x` format):**

  * Encoded as length.width.height, always in decimal format.
  * Units are **context-dependent**:

    * **Aerial/Flight contexts**: Feet (ft) default
    * **Orbital/Space contexts**: Meters (m) default
    * **Ground/Submerged contexts**: Meters (m) default
  * **No units explicitly included in notation**, inferred by context.

**Examples:**

* `+120.30.12`
  *Object/platform larger than observer's own, dimensions 120×30×12 (feet or meters contextually determined).*

* `-10.8.2`
  *Object/platform smaller than observer’s own, dimensions 10×8×2.*

* `=35.20.10`
  *Object/platform exactly matches the observer’s own platform dimensions.*
